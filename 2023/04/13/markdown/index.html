1.标题

1.1 = 和 - 的使用

使用 = 和 - 标记一级和二级标题

= 和 - 标记语法格式如下：

我展示的是一级标题
=================

我展示的是二级标题
-----------------

效果如下：
我展示的是一级标题

我展示的是二级标题

1.2 # 的使用

使用 # 号标记
使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。类型如下：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题
